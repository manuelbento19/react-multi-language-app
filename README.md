## 💼 Tecnologias utilizadas

Para o desenvolvimento deste site utilizei as seguintes tecnologias:
- React.js
- SASS
- React Icons
- i18next

## 🚀 Getting started

### 1. Install

```bash
npm install
```

or

```bash
pnpm add
```

or

```bash
yarn add
```
### 2. Run

```bash
npm run start
```

or

```bash
pnpm start
```

or

```bash
yarn start
```
